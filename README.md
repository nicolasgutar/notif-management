# Notification Management System

This repository contains the source code for the Notification Management System, designed to handle and dispatch notifications across various channels.

## Structure

- **backend**: Contains the server-side logic, API endpoints, and services (e.g., APN, Email).
- **frontend**: Contains the client-side application for managing notifications.

## Getting Started

Please refer to the README files within the `backend` and `frontend` directories for specific setup and running instructions.
